bpDatetimepicker
directive in module datetimepicker

Description

  • 时间组件
  • 提供单个时间或者范围时间

Usage

as element:
<bp-datetimepicker
       type="{String}"
       model="{String}"
       start-model="{String}"
       end-model="{String}"
       min="{String}"
       max="{String}"
       start-view="{String}"
       min-view="{String}"
       minute-step="{Number}"
       model-type="{String}"
       start-disabled="{Boolean}"
       end-disabled="{Boolean}"
       disabled
       placeholders="{Array}"
       separator="{String}">
</bp-datetimepicker>

Parameters

ParamTypeDetails
typeString
  • binding symbol is @, 是单个时间组件还是时间范围组件
modelString
  • binding symbol is =?, 单个时间值
startModelString
  • binding symbol is =?, 范围时间组件，开始时间值
endModelString
  • binding symbol is =?, 范围时间组件，结束时间值
minString
  • binding symbol is <, 最小时间限制
maxString
  • binding symbol is <, 最大时间限制
startViewString
  • binding symbol is @, 开始时间视图
minViewString
  • binding symbol is @, 最小时间视图
minuteStepNumber
  • binding symbol is <, 分钟间隔
modelTypeString
  • binding symbol is @, 显示的格式 YYYY-MM-DD 还是其他，默认到秒
startDisabledBoolean
  • binding symbol is <, 开始时间Disabled
endDisabledBoolean
  • binding symbol is <, 结束时间Disabled
disabledBoolean
  • binding symbol is <, Disabled状态
placeholdersArray
  • binding symbol is <, placeholders
separatorString
  • binding symbol is @, 范围时间分隔符，默认'至'

Example

<bp-datetimepikcer
 model="vm.model"
></bp-datetimepkcer>

this.model = '';

<bp-datetimepikcer
 type="range"
 start-model="vm.startModel"
 end-model="vm.endModel"
></bp-datetimepkcer>

this.startModel = '';
this.endModel = '';
