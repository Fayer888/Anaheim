bpCheckboxGroup
directive in module checkbox

Description

顾名思义, 这是对bpCheckbox组件的二次包装, 通过model传进来一个对象数组, 来批量渲染bpCheckbox

Usage

as element:
<bp-checkbox-group
       disabled
       model="{Array.<Object>}"
       inline="{Boolean}">
</bp-checkbox-group>

Parameters

ParamTypeDetails
disabledBoolean
  • binding symbol is <, 禁用状态, 在组上禁用则全部禁用, 实际上是对ng-disabled的包装
modelArray.<Object>
  • binding symbol is =?, 数据源, 由于是双向绑定的逻辑会去修改数据源上的checked属性
inlineBoolean
  • binding symbol is @, 是否横着排

Example

<bp-checkbox-group model="vm.groupDataList"></bp-checkbox-group>

在实际的controller或任何数据源里面有这样一个对象数组 

groupDataList = [
  {label: '邢老师不要脸', checked: true},
  {label: '老师不要脸'},
  {label: '师不要脸'},
  {label: '不要脸'},
  {label: '要脸', disabled: true},
  {label: '脸'}
];
